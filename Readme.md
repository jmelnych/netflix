# Movies Netflix app

A project of React Mentorship Program

## Get started

### Fetch submodule repo

A server-part of the project
<pre>
git submodule init
git submodule update
</pre>

### Install server dependencies

<pre>
cd MoviesAPI.ReactJS
npm i
</pre>

### Run server 
<pre>
npm start
</pre>

### Install client dependencies

<pre>
cd client
npm i
</pre>

### Run client 
<pre>
npm start
</pre>
